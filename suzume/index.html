Rhythm Game

Dancing☆Onigiri HTML5版を制作中です。

動作確認用の譜面として 夜雀の歌声 (a_hisa / goe×Cah-m) を使っています。

操作方法

  1. ゲームが始まると、音楽に合わせて矢印やおにぎり(？)が流れてきます。
  2. 貴方も音楽に合わせて、キーボードの対応するキーを押してみましょう。
  3. 対応するキーは、左から順に S, D, F, Space, J, K, L です。
  4. タイミングよくキーを押して矢印やおにぎりを消すと、より良い判定が得られます。
  5. 最後までプレイすると、貴方の得点が見られます。
  • ゲームが始まったあとも譜面の流れる速度を変えられます。Tab キーで加速、Shift キーで減速します。
  • 途中でゲームをやり直すときは Backspace キーを、やめるときは Delete キーを押します。
